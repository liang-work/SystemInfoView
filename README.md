# 一个使用Python编写的系统信息查看工具

# SystemInfoView - A System Information Viewer written in Python

使用方法：

1. 确保已安装Python 3.x
2. 安装依赖：`pip install -r requirements.txt`
3. 运行程序：`python main.py`

功能包括：

1. 查看系统信息（CPU、内存、磁盘、网络等）
2. 实时监控系统性能（CPU、内存、磁盘IO、网络等）
3. 提供系统配置信息（如系统类型、处理器架构、操作系统版本等）
4. 支持自定义配置和插件扩展
   欢迎贡献代码和提出建议！

## 注意：某些功能可能不完善。

理论上程序支持android，不过需要自己打包。
